# @artdeco/deploy

`@artdeco/deploy` is a new Node.js npm package.

Deploy a website in the cloud.

## `artdecoDeploy():void`

Call this function to get a result you want.

```js
const artdecoDeploy = require('@artdeco/deploy')

artdecoDeploy()
```
